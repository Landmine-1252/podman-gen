Podman + Quadlet Builder

Build a real Quadlet .container file directly from the form, with an optional companion podman run command when you want it. The page stays easy to host on GitHub Pages or GitLab Pages because it uses plain local HTML, CSS, and JavaScript assets.

Quadlet-first output

Systemd-managed containers are generated as real [Container] units instead of wrapping a shell-style command inside ExecStart=.

Registry auto-update caveat

registry auto-update needs a fully qualified image reference, so the generator normalizes plain Docker Hub names to docker.io/....

Rootless by default

Install hints default to systemctl --user and the rootless Quadlet search path. Switch to rootful if that matches your host.
